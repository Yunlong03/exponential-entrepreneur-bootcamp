# Dragon — China Business Services Website

A modern, dark-themed website for Dragon, a China business services company connecting European companies with on-the-ground experts across China and Hong Kong.

Built with React + Tailwind CSS, generated via Hostinger Horizons, fine-tuned in Cursor.

---

## Pages

- **Homepage** — Hero section (main headline, tagline, call-to-action), overview of services, value proposition
- **Services** — Detailed list: sourcing, quality control, market studies, fair attendance, fractional China rep, helping Chinese SMEs go global
- **About** — The network approach, cities covered (Hong Kong, Shenzhen, Ningbo, Shanghai), team philosophy
- **Contact** — Contact form + email capture with lead magnet ("5 Critical Mistakes European Companies Make When Sourcing in China")

## File Structure

```
Horizon1/
├── apps/
│   ├── web/                    ← YOUR SITE CODE
│   │   ├── src/
│   │   │   ├── pages/          ← Page content (HomePage.jsx, ServicesPage.jsx, etc.)
│   │   │   ├── components/     ← Reusable pieces (navbar, footer, buttons, cards)
│   │   │   ├── index.css       ← Global styles (color palette, fonts, base spacing)
│   │   │   ├── App.jsx         ← Main app file (connects all pages together)
│   │   │   └── main.jsx        ← Startup file (boots React — don't touch)
│   │   ├── index.html          ← Entry point (mostly empty, React fills it)
│   │   ├── tailwind.config.js  ← Tailwind CSS configuration (custom colors, fonts)
│   │   ├── package.json        ← Libraries this project depends on
│   │   └── postcss.config.js   ← CSS processing tool (don't touch)
│   └── pocketbase/             ← Backend database for email capture (not running locally yet)
├── node_modules/               ← Downloaded libraries (generated by npm install — never commit)
├── package.json                ← Root project configuration
├── package-lock.json           ← Exact library versions (don't touch)
├── .nvmrc                      ← Node.js version this project needs
└── .version                    ← Project version number
```

**Where you work day-to-day:** `apps/web/src/pages/` (content), `apps/web/src/components/` (shared elements), `apps/web/src/index.css` (colors and styling).

## How to Run

```bash
# First time only — install dependencies
cd Horizon1
npm install

# Start the local development server
cd apps/web
npm run dev

# Open in browser
# http://localhost:3000
# On phone (same wifi): http://YOUR-LOCAL-IP:3000
```

Stop the server with **Ctrl+C**.
